Chapter 5.8: Communicating with the Player - The HUD Module

PCapture-Lib lets you easily display text and hints on the player's screen (the Heads-Up Display, or HUD).

  • HUD.ScreenText: Creates text anywhere on the screen.
  • HUD.HintInstructor: Creates an in-world hint that can point to an entity.

Both use a "builder pattern," allowing you to chain settings together.

Example: Welcome message and button hint.

function OnPostSpawn() {
    // Display text at the center of the screen (0.5x, 0.5y) for 5 seconds.
    HUD.ScreenText(Vector(0.5, 0.5, 0), "Welcome to Test Chamber X", 5)
        .SetColor("0 255 255") // Make it cyan
        .SetFadeIn(0.5)
        .SetFadeOut(0.5)
        .Enable() // This final command displays it.
        
    local button = entLib.FindByName("start_button")
    if (button) {
        // Create a hint that will stay for a very long time.
        HUD.HintInstructor("Press the button to begin", 9999)
            .SetPositioning(0, button) // 0 = at the entity's position
            .SetColor("255 255 0") // Make it yellow
            .Enable()
    }
}
